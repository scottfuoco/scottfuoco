Hi 👋 My name is Scott Fuoco
============================

Full Stack Web Developer and Team Lead
--------------------------------------

I enjoy developing websites and applications primarily using React, Next.js, and Express/Fastify backends. I've worked on a variety of projects, from interactive touch-screen displays to large crypto marketplaces that have generated millions of NFTs. Recently, I have had the opportunity to work on a fun AI powered app. The application used AI APIs to build an internal tool that generates demo websites and hosts each one under a subdomain. This was a really interesting project that involved generating content, themes, and layouts, extracting data from documents, ranking images, and more all using various AI tools and APIs.

* 🌍  I'm based out of Vancouver, BC, Canada
* ✉️  You can contact me at [thisisscoie@scocodes.com](mailto:thisisscoie@scocodes.com)
* 🧠  I'm currently tinkering with AI, e-commerce, and setting up a home lab

Every year or so I try NeoVim but now I'm completely under Cursor's spell 🪄🥲
